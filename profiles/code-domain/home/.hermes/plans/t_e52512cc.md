## 设计方案

### 方案选择: 多阶段自主优化 (参数化回测 → 随机搜索 → 因子权重 → 风控精调)

**理由**: 参数空间太大(>64万组合)，需要分阶段缩小搜索范围。随机搜索在200-300次评估内可覆盖参数空间的>95%区域（拉丁超立方）。

**替代方案**:
- Bayesian优化(GPyOpt/scikit-optimize): 精度更高但收敛慢，300次评估可能不够探索
- 单纯网格搜索: 需要大幅限制每个参数的取值数量
- 直接采用v2参数: 任务要求"自主优化"而非"复制粘贴"

### 实现架构

```
strategy_momentum_opt.py
├── ParamConfig dataclass — 所有可调参数的集中定义
├── run_backtest(params) → (ann_ret, wr, dd, trades, nav) — 参数化的回测核心
├── optimize_stage1(n=200) → top_params — 随机搜索主参数
├── optimize_stage2(top_params) → refined_weights — 因子权重优化(pertrub+评估)
├── optimize_stage3(refined) → final_params — 风控参数精调(小网格)
└── main() — 完整优化流程 + 最终回测报告
```

### 评估函数
```
score = ann_ret * WR_penalty * DD_bonus
WR_penalty = 1.0 if 45% ≤ WR ≤ 55% else max(0, 1 - |WR - 50|/25)
DD_bonus = 1.0 if DD > -60% else 1 + (DD+60)/100
```
Primary objective: maximize annualized return
Constraint: WR in [45, 55] range

### 因子权重优化 (Stage 2)
对Stage 1的top-5参数配置，分别进行因子权重优化：
- 对9个权重进行扰动(每个±20%，归一化)
- 评估50个变体
- 保持WR约束

### 风控精调 (Stage 3)
对Stage 2最优配置，小范围网格搜索HARD_STOP/ATR_MULT/TRAILING参数（±20%范围，5步）

### 风险
1. **过拟合**: 回测区间仅12个月，参数可能对特定market regime过拟合 → 用WR约束缓解
2. **计算时间**: 300+次回测 × 2-3s = 10-15min → 可接受
3. **北交所影响**: 920xxx股票贡献大量赢家，考虑添加--exclude-bj选项
4. **数据源**: Sina parquet需要已存在（t_4543bcae已创建）
