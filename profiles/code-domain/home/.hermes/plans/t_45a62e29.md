**目标**: 统一数据获取接口，自动缓存

```python
class DataLoader:

```

**数据流**:
1. MCP stock-sdk (get_history_kline / get_fund_flow tools)
2. → 标准化为统一Schema (英文列名)
3. → Parquet缓存到 cache/kline_{symbol}_{start}_{end}.parquet
4. → 返回pd.DataFrame

**缓存键**: `hashlib.md5(f"{symbol}_{start}_{end}".encode()).hexdigest()[:12]`
**回退**: MCP不可用时 → fallback_resolver.load_kline → raise

# Factor evaluation

def neutralize(factor: pd.Series, industry: pd.Series,

    """行业+市值中性化 (残差法)"""

**RankIC实现**:
```python
from scipy.stats import spearmanr
ic, _ = spearmanr(factor.rank(), forward_return.rank())
```

**中性化实现**:
```python
# OLS residual: factor ~ industry_dummies + log(market_cap)
X = pd.concat([pd.get_dummies(industry), np.log(market_cap)], axis=1)
residuals = factor - X @ np.linalg.lstsq(X, factor)[0]
```
