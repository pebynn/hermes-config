# A股每日复盘写作域 - 使用指南

## 域概述

自动完成A股内容创作全流程：

**交易日每日复盘**：
```
收盘(15:00) → 数据采集 → 量化分析 → AI写作 → 草稿箱 → 人工审核(次日发布)
```

**周末周总结**：
```
周六日15:30 → 分析本周数据 → AI生成周总结 → 18:00推送通知
```

- 周六日18:00：微信推送通知，提醒查看周总结


- 深度分析本周最热门方向（1-2个板块/题材）
- 包含本周行情回顾、热点分析、下周展望

## 文章模板

### 每日复盘（5章节）

复盘文章固定5个章节：
1. **大盘回顾** — 指数表现+量能+涨跌家数
2. **资金风向标** — 北向+主力+行业轮动
3. **热点解读** — 板块涨跌+龙头点评
4. **技术看盘** — 支撑/压力+关键信号
5. **明日策略** — 方向+关注+仓位

附：风险提示 + AIGC标识

### 周总结（4章节）

周总结文章包含4个章节：
1. **本周行情回顾** — 指数周表现+资金流向+量能变化
2. **本周最热方向深度分析** — 为什么热+龙头股+后续逻辑+数据支撑
3. **下周展望** — 方向能否延续+潜在风险+关键观察点
4. **操作建议** — 关注个股/仓位管理/止盈止损策略

字数：1500-2500字，风格：更深度的分析，更注重趋势判断

## 配置调整

```bash
vim ~/.hermes/profiles/writing-domain/config.yaml
```

关键参数：
- `model.temperature` — 创造性（0.7复盘更严谨，0.8更灵活）
- `data_collection.schedule_time` — 采集时间（默认15:30）
- `content_generation.daily_output_limit` — 每日文章数（默认1篇）

## FAQ

**Q: 非交易日怎么办？**
A: 自动跳过数据采集，不生成复盘文章。

**Q: 数据不准确怎么办？**
A: 数据验证环节会标记异常值；AKShare+东方财富双源校验。

**Q: 复盘文章几点能好？**
A: 16:30左右完成写作，17:00同步到草稿箱。

**Q: 能自定义文章风格吗？**
A: 可以修改content_generation的temperature和prompt模板。
