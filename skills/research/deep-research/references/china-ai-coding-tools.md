# China-Accessible AI Coding Tools (2026)

Constraint: Cursor, Claude Code, GitHub Copilot are blocked or unreliable behind China's Great Firewall without VPN. This reference captures the domestic alternatives that work without proxy.

## Zero-Cost Personal Stack (Recommended)

| Tool | Vendor | Model | VS Code Extension | Strengths |
|:--|:---|:---|:--|:--|
| **通义灵码** (Tongyi Lingma) | Alibaba | Qwen 2.5-Coder | `TONGYI Lingma` | Java/Go backend strongest, Alibaba Cloud ecosystem |
| **CodeGeeX** | Zhipu AI / Tsinghua | GLM | `CodeGeeX` | Open-source, best Chinese comprehension |
| **Fitten Code** | Feishi Tech | Proprietary | `Fitten Code` | Fastest autocompletion speed |
