# 量化策略验证模式

## 核心原则

**验证优先：任何理论信号必须先独立回测，有效才纳入系统。**

不盲信理论（缠论/波浪理论/江恩等），不盲信论文，不盲信平台回测。每个候选信号层必须独立验证IC和胜率。

## 验证流程

```
理论/论文/平台策略
    ↓
简化独立回测（30只 × 2年，纯pandas，5分钟）
    ↓
IC绝对值 > 0.05 且 胜率 > 52%？
    ├── 是 → 纳入系统设计，完整管线开发
    └── 否 → 砍掉或降级为"辅助过滤"
```

## 实现技巧

- 不用czsc做批量回测（太慢），纯pandas/numpy简化版（<5分钟）
- 对比随机入场基线（500次取均值）
- 多时间窗口验证衰减（T+5/10/20/40）
