# Layer 3: 量价智能验证 — 资金流数据替代方案

## 核心发现

AKShare moneyflow（东方财富"主力净流入"）已被多份实证证明不可靠：
- **BigQuant 资金流策略**：回测 -45.94%
- **新浪2019实证**：5日净流入前100名表现反而弱于后100名
- **原因**：东方财富的"主力"是算法估算（大单拆分后不可见），不是真实资金归属

## 可信替代指标（全部可从K线自算）

### 1. OBV (On-Balance Volume) 背离
- **计算**：价涨日累加成交量，价跌日累减成交量
- **信号**：价格创新高但OBV未跟随 → 顶部背离 → 卖出信号
- **优势**：不需要任何外部数据，纯量价衍生

### 2. MFI (Money Flow Index)
- **计算**：典型价=(H+L+C)/3，资金流=典型价×成交量，正负资金流14日RSI
- **信号**：MFI>80超买，MFI<20超卖，背离信号
- **优势**：量价加权，传统技术分析验证充分

### 3. VWAP 偏离
- **计算**：成交量加权均价，价格相对VWAP位置
- **信号**：价格持续>VWAP且偏离扩大→多头强势；价格跌破VWAP→转弱
- **优势**：机构参考基准

### 4. 融资融券（杠杆资金情绪，AKShare官方接口）
- **数据**：`ak.stock_margin_detail_sse()` 沪市融资融券明细
- **信号**：融资余额增速>5%/周→杠杆做多；融券余额激增→做空力量
- **可靠性**：交易所披露，真实性高

### 5. 换手率异常检测
- **计算**：日换手率 / 20日均换手率
- **信号**：比值>3→主力异动（进出判断需结合方向）；比值<0.3且价稳→底部蓄力

## Layer 3 推荐权重分配

```
Layer 3: 量价智能验证（20%）
├── 量价背离 (10%) — OBV/MFI/VWAP综合
├── 杠杆情绪 (5%)  — 融资融券余额变化
└── 换手异常 (5%)  — 异常换手检测
```
