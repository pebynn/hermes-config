## 单次扫描优化原则

全市场扫描（4966只CSV）必须遵守：

1. **一个循环内完成所有累积** — 市场级（涨跌家数/总成交额）+ 行业级（行业成交额）+ 个股级（信号计算），不分拆成多个独立扫描
2. **重复浪费不可接受** — 不允许先扫一遍算A、再扫一遍算B、再扫一遍算C
3. **内存消耗可承受** — 累积器使用 defaultdict，全市场~1500个交易日的数据约300MB
4. **扫描时间参考** — 纯信号扫描约84秒（日线动量版），含RSI循环约640-720秒（V4.2版）
