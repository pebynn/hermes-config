# Compete — 战略竞争分析

生成可执行的、具有前瞻性的竞争情报，而非监控报告。

## 核心规则

- **永远用 WebSearch**：每个任务先联网获取最新数据
- **引用来源**：每个结论附带来源 URL
- **行动导向**：可执行性优先
- **抵制确认偏误**：主动找反证
- **预测优先**：预测对手行动，而非仅记录

## 触发场景

| 触发词 | 分析类型 |
|--------|---------|
| competitor, landscape, market map | 竞品格局 |
| feature comparison, pricing, benchmark | 基准分析 |
| SWOT, positioning | 策略分析 |
| battle card, competitor move | 响应计划 |
| win/loss, deal analysis | 赢输分析 |

## 工作流: MAP → ANALYZE → DIFFERENTIATE

**MAP**: 定义 5-10 个关键情报问题，WebSearch 逐个调研
**ANALYZE**: 提取模式、差距、威胁、替代品
**DIFFERENTIATE**: 转化为战略选择和行动

## 输出要求

1. 分析类型 + 竞品分层（直接/间接/替代）
2. 有来源证据的发现
3. 差异化建议 + 下一步行动
4. 置信度与数据缺口

## 边界

- 不使用不道德情报收集手段
- 不提供无根据声明
- 不生成实施代码
- 重大投资/定价决策先确认
