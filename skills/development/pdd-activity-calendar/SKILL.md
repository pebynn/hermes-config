## 三、活动报名策略

### 3.1 报名时间轴（倒推法）

```
活动上线日 ──────────────────────────────────────── 今天
    ↑              ↑              ↑              ↑
  上线        审核通过       提交报名       开始准备
 (D-day)    (D-3~5天)     (D-7~10天)    (D-14~21天)
```

| 阶段 | 距活动 | 动作 |
|:----|:------|:-----|
| **准备期** | D-14~21天 | 选品、定价测算、库存盘点、主图优化、准备报名素材 |
| **报名期** | D-7~10天 | 商家后台提交报名，填写活动价和库存 |
| **审核期** | D-3~5天 | 系统审核+运营审核，关注审核反馈 |
| **排期确认** | D-1~3天 | 确认资源位排期，检查活动价和库存 |
| **活动上线** | D-day | 活动生效，监控流量和转化 |
| **活动复盘** | D+1~3天 | 流量/销量/转化率/ROI分析 |

### 3.2 价格策略（活动定价铁律）

**核心原则：活动是流量入口，不是利润中心。**

| 活动类型 | 降价幅度 | 定价策略 |
|:--------|:--------|:--------|
| 限时秒杀 | 日常价×0.85~0.9 | 覆盖成本底线即可，靠量取胜 |
| 百亿补贴 | 全网最低价（比淘宝低10%+） | 平台补贴部分差价，商家让利有限 |
| 9.9特卖 | ¥9.9-29.9区间 | 只适合引流款（成本<¥20） |
| 断码清仓 | 日常价×0.5~0.7 | 清库存回笼资金，不求利润 |
| 大促（618/双11） | 日常价×0.8~0.85 | 预留推广费空间，保整体ROI>1.5 |
| 领券中心 | 券面额≥标价×50% | 通过优惠券隐性降价，不直接改标价 |

**活动定价倒算法**：
```
活动价底线 = 成本 + 快递费(3-5元) + 包装费(0.5-1元) + 退货损耗(3-5元) + 最低利润(3-5元)
```
举例：一件成本30元的夏装套装 → 活动价底线约¥41.5，秒杀价可设¥45-49保本微赚。

### 3.3 库存准备策略

| 活动类型 | 备货量（×日均销量） | 说明 |
|:--------|:------------------|:-----|
| 限时秒杀 | 5-10倍 | 秒杀流量爆发力强，断货=损失流量+排名下降 |
| 9.9特卖 | 3-5倍 | 走量为主，多SKU分摊 |
| 大促（618/双11） | 10-20倍 | 年度最大流量，错失=错失全年 |
| 母亲节/女神节 | 5-8倍 | 中老年女装需求集中爆发 |
| 断码清仓 | 现有库存全部 | 清尾货，不补货 |

**库存红线**：
- 爆款SKU库存至少留30%给活动后的自然流量承接
- 加大码（XXL-5XL）库存占总库存30%以上，活动中最易断货
- 活动前3天完成补货，活动期间断货=报名作废且可能被处罚

### 3.4 DSR门槛维护

所有核心活动的硬性门槛都是 DSR ≥ 4.7，三项（描述/物流/服务）需同步达标。详见 `ec-ops-daily` 的DSR维护章节。

## 五、活动与推广的协同策略

### 5.1 活动前中后的推广节奏

```
活动前（D-7~D-1）────→ 活动中（D-day）────→ 活动后（D+1~D+7）
   蓄水期                  爆发期                承接期
```

#### 蓄水期（活动前 7 天）

| 动作 | 推广配合 | 目的 |
|:----|:--------|:-----|
| 主图加「即将参加XX活动」标签 | 多多搜索小幅加价20% | 提高收藏加购 |
| 优惠券预热（满59减5→满99减15） | 多多场景低预算试投 | 预热活动氛围 |
| 评价积累（追评晒图引导） | — | 活动期高转化需要好评价背书 |
| 库存+主图+详情页全部到位 | — | 避免活动开始后调整 |

#### 爆发期（活动中）

| 动作 | 推广配合 | 预算建议 |
|:----|:--------|:--------|
| 活动上线立即监控流量和转化 | — | — |
| 前2小时转化好→追加推广预算 | 多多搜索提价30-50% | 日预算翻倍 |
| 关注竞品是否跟价 | — | — |
| 准备返场素材 | 多多场景全开 | 50-100元/日 |

> **黄金法则**：活动前2小时是流量最大的窗口期。如果这2小时转化率低于1.5%，立即检查价格/主图/评价是否有问题。

#### 承接期（活动后 7 天）

| 动作 | 推广配合 | 目的 |
|:----|:--------|:-----|
| 活动结束后自然流量会下降30-50% | 全站推广接管（需50+好评） | 维稳销量不下滑 |
| 整理活动期间的评价（差评2小时内处理） | — | 活动积累的评价影响搜索权重 |
| 挑选活动中表现TOP 3商品 | 搜索继续保留，ROI<1.5的词暂停 | 集中资源 |
| 返场价（比活动价高5-10%） | 多多场景低成本覆盖 | 收割犹豫用户 |

### 5.2 活动×推广工具组合矩阵

| 活动类型 | 冷启动（1-2周） | 活动期 | 活动后 |
|:--------|:-------------|:------|:------|
| 限时秒杀 | 多多搜索长尾词 | 搜索提价50% + 场景全开 | 全站推广ROI 2.5-3.5 |
| 大促（618/双11） | 搜索蓄水+场景预热 | 全站推广接管+搜索追加 | 场景低价覆盖+全站维稳 |
| 9.9特卖 | 不开推广（测款） | 不开推广（看自然转化） | 转化好的→搜索冷启动 |
| 母亲节 | 搜索「妈妈装」「母亲节礼物」词 | 搜索出价翻倍+场景定向50+女性 | 全站推广过渡 |
| 领券中心 | 多多进宝（达人推券） | 场景定向竞品粉丝 | 搜索维持 |
| 断码清仓 | 不开推广 | 多多进宝佣金提高至25% | 低价场景扫尾 |

### 5.3 推广工具阶段顺序（不可跳过）

```
活动报名拿基础销量 → 多多搜索精准词 → 秒杀/大促冲量 → 全站推广接管 → 场景低成本收割
    (第1-2周)        (第2-3周)         (第3-4周)        (第5-8周)        (第9周+)
```

**关键：有50+好评后才启动全站推广。** 新品没有评价积累前，全站推广ROI难以达标。

## 六、特殊活动机制详解

### 6.1 洪水链接（品牌旗舰店专属）

大促期间（618/双11）部分受邀品牌旗舰店可通过"洪水链接"以超低门槛直进大促内场：

| 链接类型 | 比价范围 | 权益 |
|:--------|:--------|:-----|
| A链接 | 仅需旗舰店同款同价，不需全网最低价 | 大促官方补贴+全站搜推加权 |
| B链接 | 仅需旗舰店同款同价 | 同上 |
| H链接 | 站外只不比淘C，站内正常比价 | 可生效至元旦大促，跨活动衔接 |

**获取条件**：购买大促品牌包+完成站外官微置换+完成对标店铺自助提报。

### 6.2 降价补差（价保）

- 大促期间打标"降价补差"的商品，消费者在价保周期内若商品降价可申请差价退还
- 商家报名时若系统建议价特别离谱，通常是比价异常，可在报名处反馈或由招商在「同款比价→手动比价」提交报错

### 6.3 大促搜索池

- 门槛低、通过率高，报名后在全站维度被推荐加权
- 商品打上大促标志，在主会场和分会场个性化展示
- **全类目（除部分虚拟商品）均可报名**
- 所有大促必须报名此入口

### 7.2 活动中老年女装选品检查清单

报名活动前，确认以下事项：

- [ ] 价格符合活动区间（秒杀≤日常价×0.9，大促≤日常价×0.85）
- [ ] 库存满足活动量（秒杀≥日均5倍，大促≥日均10倍）
- [ ] DSR三项≥4.7（低于则先修复，参见 `ec-ops-daily`）
- [ ] 主图有视频版本（+15-20%加权曝光）
- [ ] 标题含对应活动/节日词（如"母亲节礼物""618大促"）
- [ ] 加大码（XXL-5XL）库存充足（占比≥30%）
- [ ] 好评数≥20条（大促会场对好评有门槛要求）
- [ ] 退货率<20%（超过红线会被限制报名）
- [ ] 竞品价格已监控（避免比价异常导致建议价被拉低）
- [ ] 推广预算已预留（活动期推广费预留日常的2-3倍）

## 八、风险与避坑指南

### 8.1 活动报名常见踩坑

| 坑 | 表现 | 规避方法 |
|:--|:----|:--------|
| 比价异常 | 系统建议价低得离谱 | 在「同款比价→手动比价」提交报错 |
| 库存不够 | 活动中断货，报名作废 | 活动前3天完成补货，留30%库存给活动后 |
| DSR不达标 | 无法报名核心活动 | 日常维护DSR（每3天检查一次分数趋势） |
| 价格过低亏本 | 秒杀价覆盖不了成本 | 用活动定价倒算法（第三章）先算清楚底线 |
| 评价太少 | 大促会场不给流量 | 活动前至少积累20条好评 |
| 低价引流处罚 | 起标价与SKU差价>50% | SKU差价控制在30%以内 |
| 活动后流量断崖 | 自然流量下降30-50% | 提前准备好承接方案（全站推广+返场价） |
| 多活动冲突 | 同一商品报名多个活动导致比价混乱 | 一个商品同一时间只参加一个活动 |

### 8.2 政策风险关注点

- 2025年起「仅退款」政策松动，「自动跟价」可能被调整 → 关注活动报名时的比价规则变化
- 「千亿扶持」计划持续推进 → 留意平台对中小商家的活动专项扶持
- 双十一「不再要求全网比价」政策（2025年已实施）→ 更多品牌商家获活动入场券，竞争加剧
