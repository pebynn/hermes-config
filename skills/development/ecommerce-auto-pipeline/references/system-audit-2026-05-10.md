### 阻塞原因 (由深到浅)

**1. beast-core checkbox 完全阻塞 (P0)**
- React 合成事件不认模拟点击
- 6层攻击手段全失效: fiber/CDP/text-locator/DOM hack/PyAutoGUI
- 尺码 checkbox DOM 在 React production 模式下不渲染标准 input/checkbox
- 根源: React SyntheticEvent 经 ShallowEventPool+EventPropagation 重写事件流

**2. Playwright pipe 模式不工作**
- 系统默认 `--remote-debugging-pipe` 在 Wayland/Xwayland 下挂起
- `--remote-debugging-port=0` 修补方案在 Node.js Playwright 中仍用 pipe 传输层
- 修补文件: `~/.local/share/pip/*/playwright/driver/package/lib/server/chromium/chromium.js`
- 当前访问不能无头启动

**3. 缺少 PDD API 凭证**
- `~/.pdd_api_config.json` 不存在
- 环境变量 `PDD_CLIENT_ID` 未设置
- `PDD_CLIENT_SECRET` 和 `PDD_ACCESS_TOKEN` 已设但客户端未配置

**4. 缺少登录会话**
- `~/.pdd_auth.json` 不存在（需要 `pdd_login_v2.py --headed` 手动过滑块）
