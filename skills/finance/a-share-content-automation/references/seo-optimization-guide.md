# 公众号SEO优化指南

## 背景
2026年，微信搜一搜月活8亿+。公众号文章通过搜一搜获得自然搜索流量的机会巨大。原有复盘文章的标题和内容结构缺乏SEO意识，导致搜索不可见。

## 核心策略：3层优化

### 1. 标题公式
```
[情绪钩子] + [具体数字] + [核心关键词] + [日期]
```

**公式拆解：**
- 情绪钩子：反问/感叹/对比/悬念（如"跑了319亿却在涨"、"该慌还是该跟"）
- 具体数字：涨跌幅、成交额、涨停家数等
- 核心关键词：搜一搜高频词（上证指数/今日股市/大盘分析/A股复盘/主力资金/涨停）
- 日期：明确日期增加时效性

**好的例子：**
- ✅ "主力资金跑了319亿，A股今天却在涨？散户该慌还是该跟？"
- ✅ "今日复盘：科创50涨X%，涨停XX家，这个板块最火"
- ✅ "A股今天发生了什么？5月9日三大指数涨跌互现"

**坏的例子：**
- ❌ "2026-05-09 A股每日复盘"（无情绪钩子、无搜索词）
- ❌ "三大指数集体收涨，A股要变天了？"（太模糊、数据不具体）

### 2. 搜一搜元描述（前100字）
文章前100字会作为搜一搜搜索结果摘要展示。必须包含：
- 当日核心数据（涨跌幅/成交额/涨停数）
- 至少2-3个核心关键词
- 一句话说清当日市场状态

**模板：**
```
今天A股三大指数[涨跌互现/全线上涨/全线下跌]，[上证/科创50]收报XXXX点涨X.XX%，
两市成交额XXXXX亿[放量/缩量]。涨停XX家跌停XX家，主力资金[净流入/流出]XXX亿。
[最强板块/最大亮点一句话描述]
```

### 3. 正文关键词布局
- 前300字自然融入3-5个核心关键词
- 小标题中含关键词
- 关键词密度控制在1-3%（非堆砌）
- 自动从数据提取关键词函数：`_compute_seo_keywords()`

**自动关键词提取逻辑：**
| 数据维度 | 提取的Tag | 示例 |
|:---------|:----------|:-----|
| 上证指数数据 | 上证指数、今日股市 | 上证+0.48% → tag: 上证指数 |
| 科创50数据 | 科创50 | 科创50+1.32% → tag: 科创50 |
| 涨停家数 | 涨停 | 涨停99家 → tag: 涨停 |
| 主力资金 | 主力资金 | 净流出319亿 → tag: 主力资金 |
| 最强板块 | [板块名] | 通信设备+3.2% → tag: 通信设备 |

## 互动引导

在文章末尾增加CTA（Call to Action）：

```markdown
### 💬 互动时间
**今日互动：** 你今天的操作怎么样？看好哪个板块？欢迎在评论区留言交流 👇

**📌 还没关注？** 点一下关注，每天收盘后第一时间收到复盘推送！
**🔍 想知道某只股票的主力资金流向？** 回复「股票代码」即可查询（如: 600519）
```

## 小绿书短内容

小绿书是微信公众号的短图文功能（类似小红书），不占用每日群发次数。

### 内容选择策略
从每日数据中自动提取3个不同类型的亮点：
1. **资金类**（优先级最高）：主力资金大额进出 → 散户关注度最高
2. **涨跌停类**：涨停/跌停家数异常时 → 情绪指标
3. **指数/板块类**：大涨或大跌的指数/板块
4. **量能类**：成交额异常放量/缩量

### 生成模板
每条约200字 + 1张相关图表。5种内容类型：
| 类型 | 图标 | 适用场景 |
|:-----|:-----|:---------|
| index | 📈📉 | 指数涨跌幅>0.3% |
| fund | 💰 | 主力资金净流入>50亿或净流出>50亿 |
| limit | 🎯 | 涨停或跌停家数突出 |
| sector | 🔥 | 板块涨跌幅>1% |
| volume | 📊 | 成交额>20000亿 |

## 效果衡量指标
- **搜一搜来源流量**：公众号后台→内容分析→搜一搜来源阅读数
- **文章打开率**：从现有1-2%/篇 → 目标5%+
- **单篇阅读量**：从1-2/篇 → 目标30-100（第1周）→ 200-500（第2-3周）
- **互动率**：在看+点赞+评论 > 3%
