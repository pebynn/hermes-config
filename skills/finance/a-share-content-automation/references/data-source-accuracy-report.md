# A股数据源准确性报告

基于 2026-05-05 深度调研：AKShare/Tushare Pro/Baostock/EastMoney直连/Wind/Choice。

## 核心结论

### 可以做到100%准确的维度
- 指数收盘价/涨跌幅（AKShare `stock_zh_index_daily_em` — 东方财富聚合交易所数据）
- 两市合计成交额（上证+深证预计算）
- 涨跌停列表（过滤北交所/IPO/ST/退市后）

### 只能接近100%（估算值）的维度
- 主力资金（基于L1逐笔成交分类，≥500万=超大单，100-499万=大单，20-99万=中单，<20万=小单）
- 行业资金流向（同上，且板块分类标准不同导致偏差）

### 致命误区
- `stock_board_industry_hist_em()` — 实测数据范围仅 2021-12 ~ 2022-04，不可用于当前数据
- `stock_board_industry_index_ths()` — 2020-01 ~ 2024-01，也不可用
- `stock_board_concept_hist_em()` — 2022-01 ~ 2022-11，不可用

### 正确方案
板块历史数据 = 当日准时采集 + JSON存盘 + 周总结读盘。不依赖不存在的历史端点。
