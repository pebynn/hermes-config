# 周末内容策略 — 2026-05-11 实施

## 核心决策

❌ 不把周总结拆成每日生成 — 和每日复盘内容重叠
✅ 用"量化周报(周六) + 科普加量(周六日)"填补周末空白

## 周末管线时间表

```
周五 16:00 → weekly_summary.py + publish (3858ff88add6)
              └─ 含"下周关注"章节（信号引擎注入）

周六 08:00 → quant_weekly.py (bc02d5952723) — 量化信号周报
周六 08:00 → 科普: 主力资金 (9f73cbaa5f1e, no_agent)
周六 14:00 → 科普: 追涨 (d50d838746d6, no_agent)

周日 08:00 → 科普: 市盈率 (d403a750c641, no_agent)
周日 14:00 → 科普: 新手亏钱 (032e7102e419, no_agent)
```

周末总产出: 1篇量化周报 + 4篇科普 = 5篇

## 量化周报搬周六的理由

- quant_weekly.py 已成熟(1155行)，含 publish_draft.py 集成
- 周末流量高峰在周六上午，周日15:30基本无人读
- 从周日15:30 → 改到周六08:00，不增成本

## 科普周末加量

- generate_popular.py 支持 `--topic` 参数，自动推草稿箱
- 7个预设选题轮换: k线/基金定投/主力资金/市盈率/涨停/新手亏钱/追涨
- 统一入口: `~/.hermes/scripts/run_weekend_kepu.sh <topic>`
- 内置去重: 扫描已有草稿和已发表文章，同标题/高相似度自动跳过

## 周总结"下周关注"实现

- weekly_summary.py 新增 `compute_next_week_focus()` 函数
- 调用 signal_engine.py 获取近5日买入信号
- 解析 JSON/markdown table 两种输出格式
- 提取板块关键词 → 汇总3个方向
- 注入到 prompt 的"下周关注"章节，AI 自然融合而非照抄

## 成本

量化周报 $0.08/次 + 科普 $0.03×4 = $0.20/周末
