# 微信草稿箱 API 字段限制

微信公众号 `cgi-bin/draft/add` 对各字段有严格限制，与公众号管理后台的编辑限制不同。

## 实测限制（2026-05-05）

| 字段 | 硬限制 | 错误码 | 实测值 |
|:--|:--|:--|:--|
| title | ≤64字符 | 45003 | 15中文字安全 |
| author | ≤8字节 | 45110 | 固定"AI复盘"（4字节） |
| digest | ≤120字节(≈14中文字) | 45004 | 14中文字安全 |
| content | 无明确限制 | - | 11KB HTML 通过 |
| thumb_media_id | 必填 | 40007 | 需重新上传（token 有效期7200s） |

## 关键发现

1. **author 字段极严**: "复盘点金"(4中文字=12字节)、"复盘手"(3字=9字节) 均失败。只有 "AI复盘"(2字ASCII+2字中文=8字节) 和 "A"(1字节) 通过。

2. **digest 14中文字**: 120字节≈14中文字(UTF-8每中文字3字节)，留4字节余量。

3. **JSON 序列化**: 使用 `data=json.dumps(body, ensure_ascii=False)` 而非 `json=body`，否则中文变 `\uXXXX` 转义。

4. **Token 刷新**: token 7200s 过期，上传图片和创建草稿之间若超时需重新获取。图片的 media_id 绑定当前 token。

## errors 速查

| 错误码 | 含义 | 处理 |
|:--|:--|:--|
| 0 | 成功 | - |
| 40001 | token 失效 | 获取新 token |
| 40007 | thumb_media_id 无效 | 重新上传封面图 |
| 40164 | IP 未在白名单 | 用浏览器自动化 |
| 45003 | 标题过长 | 截断至 ≤15 中文字 |
| 45004 | 摘要过长 | 截断至 ≤14 中文字 |
| 45110 | 作者过长 | 固定 "AI复盘" |
