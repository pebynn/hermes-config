## 标题生成规则

### 每日复盘标题

**格式**：【每日复盘】日期 + 核心数据 + 热点

**示例**：
- 【每日复盘】2026-05-05 上证+1.2% 成交额破万亿，AI算力掀涨停潮
- 【每日复盘】2026-05-04 三大指数全红，北向净流入50亿，新能源卷土重来
- 【每日复盘】2026-05-03 沪指守住3200点，医药板块爆发，5股涨停

**要素**：
1. 日期
2. 指数涨跌幅（±X.XX%）
3. 关键数据（成交额/资金流/家数比）
4. 最热板块/题材（2-3字）

### 周总结标题

**格式**：【周总结】本周最热方向：[方向名称]，下一周怎么看？

**示例**：
- 【周总结】本周最热方向：AI算力，下一周怎么看？
- 【周总结】本周最热方向：新能源车，下一周怎么看？
- 【周总结】本周最热方向：半导体，下一周怎么看？

## 正文写作规则

### 数据引用规范

**严格遵循**：
- 所有数据必须有来源（AKShare/东方财富）
- 禁止编造数据或凭空猜测
- 百分比保留2位小数（如+1.23%）
- 金额保留整数（如1200亿）

**格式示例**：
```
✅ 正确：上证指数收于3234点（+1.23%），成交额8500亿，较昨日+15%
❌ 错误：上证涨了1.2%左右，成交量大概8000多亿
```

### 章节字数分配

#### 每日复盘（1500-2000字）

|------|------|------|
| 大盘回顾 | 300-400字 | 数据+量能+涨跌家数 |
| 资金动向 | 300-400字 | 北向+主力+板块轮动 |
| 热点板块 | 500-600字 | Top3板块深度分析 |
| 技术分析 | 200-300字 | 关键位置+信号 |
| 明日策略 | 200-300字 | 方向+关注+仓位 |

#### 周总结（1500-2500字）
| 章节 | 字数 | 重点 |
|------|------|------|
| 本周行情回顾 | 400-500字 | 指数周表现+资金流 |
| 最热方向深度分析 | 700-1000字 | 驱动因素+龙头+数据 |
| 下周展望 | 300-400字 | 延续性+风险+观察点 |

## 标签规范

### 每日复盘标签
```
#A股 #复盘 #股市 #投资策略 #[当日最热板块]
```

### 周总结标签
```
#A股 #周总结 #投资策略 #[方向名称]
```

**规则**：
- 固定标签在前（#A股 #复盘 #股市 #投资策略）
- 变量标签在后（最热方向）
- 最多5个标签，用空格分隔
- 标签不包含特殊字符（如+、%、,）
