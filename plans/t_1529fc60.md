## 2. 设计方案 (Design)

### 2.1 方案选择：自主闭环优化 (Self-Optimization Loop)

```
┌─────────────────────────────────────────────────┐
│              Self-Optimization Loop              │
│                                                  │
│  Phase 1: Parameter Grid Search                  │
│    ↓ (best params → baseline)                    │
│  Phase 2: Factor Weight Optimization (Bayesian)  │
│    ↓ (optimal weights → factor score)            │
│  Phase 3: Signal Filter Tuning                   │
│    ↓ (adaptive thresholds → cleaner signals)     │
│  Phase 4: Risk Control Iteration                 │
│    ↓ (exit param tuning → better RR ratio)       │
│  REPEAT from Phase 1 (warm-start with best)      │
└─────────────────────────────────────────────────┘
```

### 2.2 替代方案及淘汰理由

| 方案 | 优点 | 淘汰理由 |
|:-----|:-----|:---------|
| 纯随机搜索 | 实现简单 | 收敛慢，11维参数空间需大量样本 |
| 遗传算法 | 全局最优 | 回测每次~30s，种群迭代太慢 |
| 梯度优化 | 快速收敛 | 回测不可微，无法计算梯度 |
| **分层网格+贝叶斯 ✓** | 可解释+渐进 | 选中：分阶段优化，每阶段可独立验证 |

### 2.3 核心架构

```
OptConfig (参数空间定义)
  ├── PARAM_GRID: {param_name: [values]}
  ├── FACTOR_WEIGHTS: {factor: (min, max, default)}
  └── PHASE_ORDER: ["param_search", "factor_weights", "signal_filter", "risk_control"]

compute_factor_score(df, weights) → df['score']
  - 多因子加权: ret_5d * w1 + ret_3d * w2 + vol_ratio * w3 + rsi * w4 + ...
  - 无未来数据泄露 (所有因子lagged)

run_single_backtest(params) → {metrics dict}
  - 参数注入 → 运行完整回测 → 返回标准化指标

optimize_phase(config, phase_name) → best_params
  - Grid search (Phase 1): 粗网格→细网格渐进
  - Bayesian opt (Phase 2): scipy.optimize 或手动实现
  - Heuristic search (Phase 3/4): 基于上一阶段最优参数的局部搜索

main_optimize() → 完整闭环
  - CLI: python strategy_reversal_opt.py --optimize
  - 迭代至收敛或max_rounds
```

### 2.4 参数空间设计

PARAM_GRID = {
    "DECLINE_PREFILTER": [-0.03, -0.04, -0.05, -0.06, -0.08],  # 5 levels
    "DECLINE_QUANTILE":  [0.12, 0.15, 0.18, 0.20, 0.25],       # 5 levels

### 2.5 风险与缓解

| 风险 | 影响 | 缓解措施 |
|:-----|:-----|:---------|
| 过拟合回测区间 | 实盘失效 | 不切分训练/测试(仅回测区间)，但通过简单参数+低组合数控制 |
| 计算时间爆炸 | 网格400*30s=3.3h | 渐进式：粗网→top-N细网；并行回测 |
| 300%目标不可达 | 用户预期管理 | 在优化输出中报告理论上限，诚实面对LEV=1.0约束 |
| 因子权重不稳定 | 换区间失效 | 约束权重范围[0,1]并归一化，不过度依赖单一因子 |
