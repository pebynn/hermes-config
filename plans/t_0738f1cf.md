### 2.3 Partial Take Profit Logic (NEW)

When position reaches +8% profit:
1. Sell 50% of position at current close
2. Record partial exit as a trade with `exit_reason = "partial_tp"`
3. Remaining 50% continues with original entry_price and adjusted weight
4. Remaining portion still subject to: full TP (+15%), stop loss (-5%), max_hold (40d)
5. Trade record for remaining portion has `exit_reason` reflecting final disposition

Implementation approach:
- Add `partial_exit_taken` flag to position dict
- In exit check loop: if pnl >= 0.08 and not partial_exit_taken:
  - Record partial trade (weight * 0.5, pnl = 8%)
  - Set partial_exit_taken = True, halve weight
  - Continue tracking with same entry_price
- Full TP at 15% still triggers for remaining half
