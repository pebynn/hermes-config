## Brainstorming 记录

### MARKET_TIMING hybrid 模式详细设计

当前 `market_timing_allows(ma20, ma60, mode)` 支持 "ma_cross" / "adx" / "both" / "disabled"。
"both" 模式只检查 ma20 > ma60，未实际加入ADX。

新增 "hybrid" 模式:
```python
if mode == "hybrid":
    # Primary: golden cross (MA20 > MA60)
    if ma20 > ma60:
        return True
    # Catch: MA20 near MA60 (within 2%) AND MA20 slope positive
    if ma20_slope_5d is not None and not pd.isna(ma20_slope_5d):
        if ma20 > ma60 * 0.98 and ma20_slope_5d > 0:
            return True
    return False
```

`compute_market_proxy()` 增加:
```python
daily_ret["ma20_slope_5d"] = daily_ret["ma20"].diff(5) / daily_ret["ma20"].shift(5)
```

主循环 proxy_lookup 增加 slope 字段:
```python
proxy_lookup[row["trade_date"]] = {
    "ma20": ..., "ma60": ..., "proxy_ret": ...,
    "ma20_slope_5d": row.get("ma20_slope_5d", np.nan)
}
```

主循环调用处:
```python
market_ok = market_timing_allows(
    proxy.get("ma20", np.nan),
    proxy.get("ma60", np.nan),
    mode=MARKET_TIMING_MODE,
    ma20_slope_5d=proxy.get("ma20_slope_5d", np.nan)
)
```

### 注意事项
- market_timing_allows 签名需向后兼容（ma20_slope_5d 默认 None）
- compute_market_proxy diff(5) 需要 warmup，前几日 slope 为 NaN → 自动回退到纯金叉
- 捕捉路径仅在 MA20 在 MA60 的 2% 以内时触发，防止假反转
