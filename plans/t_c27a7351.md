### Implementation Strategy

**Approach**: The current `strategy_reversal_B.py` already has R11 baseline parameters
(MOM=6%, SECTOR=3, CAP=300K, MODE=ma_cross) but includes R14 additions (TRAILING=-3%,
VOL_RATIO=1.5, QUALITY, SCORE_DECAY). 

Two options:
A. Modify current file to R13-B (disable R14 features, keep R14 code as dead/commented)
B. Create separate strategy_reversal_B_R13.py

**Choice: Option A** — Single evolving file, parameters as config flags. R14 features
are toggled off via boolean constants. This keeps the codebase clean and the evolution
traceable.

**Changes needed to reach R13-B from current "R14" code**:
1. TRAILING_STOP_PCT: -0.03 → -0.04
2. VOL_RATIO_MIN: 1.5 → 1.0
3. QUALITY_FILTER_ENABLED: True → False
4. SCORE_DECAY enforcement: disable in the exit loop (guard with boolean)
5. Update docstring header to "R13-B"
